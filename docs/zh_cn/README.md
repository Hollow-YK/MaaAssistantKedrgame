# MAK 文档

## 简介

MAK（MaaAssistantKedrgame）是一款基于 [MaaFramework](https://github.com/MaaXYZ/MaaFramework) 的《雪松》游戏辅助工具，通过图像识别模拟常规交互操作，帮助玩家完成重复性日常任务。

本项目开源、免费，仅供学习和交流使用。

## 快速开始

### 环境要求

- 操作系统：Windows 10 及以上
- 模拟器/设备：推荐使用 `MuMuPlayer`、`雷电模拟器`，理论上支持任意支持 adb 的 Android 模拟器及 Android 设备。
- 游戏设置：必须使用 16:9 分辨率，推荐 `720P` (1280*720)

### 安装步骤

前往 [GitHub Release](https://github.com/Hollow-YK/MaaAssistantKedrgame/releases) 下载最新版本.

MAK 提供两种安装方式：

#### 安装包版（推荐）

1. 下载 `MAK-win-x64-{版本号}-Setup.exe`
2. 双击运行安装程序，按向导完成安装。安装过程可能被部分杀毒软件阻止，放行即可。若你对此有顾虑可以查看我们的全部源码以确保我们提供的内容是安全的。
3. 安装完成后双击桌面快捷方式或开始菜单中的 `MAK` 启动

#### 压缩包版（便携）

1. 下载 `MAK-win-x86_64-{版本号}.zip`
2. 将压缩包解压到纯英文目录（**注意：路径中请勿包含中文或特殊字符**）
3. 双击 `MAK.exe` 启动程序

### 初次配置

1. 启动后点击「添加任务」，选择需要执行的功能
2. 根据提示完成游戏模拟器/设备绑定
3. 点击「开始」即可运行
