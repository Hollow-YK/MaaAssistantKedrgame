# 启动游戏

让 MAK 帮你自动打开《雪松》游戏。

---

## 这个功能会做什么？

当你勾选「启动游戏」并点击开始后，MAK 会自动：

1. 在模拟器中打开《雪松》游戏
2. 等待游戏加载完成
3. 如果有资源更新，会自动等待更新完成
4. 点击屏幕进入游戏主界面

> 整个过程你什么都不用管，等着就行！

---

## 注意事项

- 确保你已经登录过模拟器中的《雪松》游戏，并且其登录状态仍有效（没有在其它设备登录，且在不久之前登录过）
- 确保模拟器已经**正常启动**，不要在模拟器关闭的状态下使用
- 游戏必须已经**安装**在模拟器中，MAK 不会帮你下载游戏
- 如果游戏需要更新资源，MAK 会等待更新完成，这个过程可能会多花一点时间，请耐心等待
- 模拟器分辨率需设置为 16:9（推荐 1280×720 或 1920×1080）
- 运行过程中**不要操作模拟器**，让 MAK 自己完成
