# Launch Game (启动游戏)

Let MAK automatically open 《雪松》 for you.

---

## What Does This Feature Do?

After you select **启动游戏** (“Launch Game”) and click **开始** (“Start”), MAK will automatically:

1. Open 《雪松》 in the emulator.
2. Wait for the game to finish loading.
3. Wait automatically for any resource updates to finish.
4. Tap the screen to enter the game's home screen.

> You do not need to do anything during this process—just wait for it to finish!

---

## Notes

- Make sure you have previously logged in to 《雪松》 in the emulator and that the login session is still valid (you have not logged in on another device, and you logged in recently).
- Make sure the emulator has **started normally**. Do not use this feature while the emulator is closed.
- The game must already be **installed** in the emulator. MAK will not download it for you.
- If the game needs to update resources, MAK will wait for the update to finish. This may take a little longer, so please be patient.
- Set the emulator resolution to 16:9 (1280×720 or 1920×1080 is recommended).
- **Do not operate the emulator** while the task is running.
