# Claim Rewards (领取奖励)

Let MAK automatically claim various in-game rewards for you.

---

## What Does This Feature Do?

When you use **领取奖励** (“Claim Rewards”), MAK automatically claims the following rewards:

### 1. Task Rewards

- **每日任务** (“Daily Tasks”) rewards: Rewards available after completing daily tasks.
- **每周任务** (“Weekly Tasks”) rewards: Rewards available after completing weekly tasks.
- **军旅成就** (“Military Achievements”) rewards: Rewards available after completing achievements.

### 2. Resource Dispatch (资源派遣)

- **Claim dispatch rewards**: Automatically collects completed **资源派遣** (“Resource Dispatch”) rewards.
- **Dispatch again**: Automatically starts another dispatch after collecting the rewards, so you do not need to do it manually each time.

---

## How Do I Configure It?

If you only want to claim certain rewards, expand the settings for **领取奖励**:

| Setting | Description |
|---------|-------------|
| **领取任务奖励** (“Claim Task Rewards”) | Master switch; when disabled, no task rewards are claimed |
| 　↳ **每日任务** (“Daily Tasks”) | Controls daily task rewards separately |
| 　↳ **每周任务** (“Weekly Tasks”) | Controls weekly task rewards separately |
| 　↳ **军旅成就** (“Military Achievements”) | Controls achievement rewards separately |
| **资源派遣** (“Resource Dispatch”) | Whether to claim dispatch rewards |
| 　↳ **再次派遣** (“Dispatch Again”) | Whether to dispatch again automatically after claiming rewards |

---

## Notes

- Before using this feature, make sure you are logged in to the game.
- **Do not operate the emulator** while the task is running.
