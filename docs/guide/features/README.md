# 功能介绍

MAK 目前提供以下功能，每项功能都可以独立使用。

你可以在 MAK 主界面点击「**添加任务**」，然后勾选你需要的功能。MAK 会按照任务列表从上到下的顺序依次执行。

---

## 功能列表

| 功能名称 | 说明 | 详细文档 |
|---------|------|---------|
| 启动游戏 | 自动打开《雪松》游戏并等待加载完成 | [查看详情](./start-game.md) |
| 领取奖励 | 自动领取日常任务、周常任务、成就、派遣等奖励 | [查看详情](./award.md) |

---

## 如何使用功能？

每个功能称为一个「任务」。你可以：

- **勾选/取消勾选**：决定要不要执行这个任务
- **拖拽排序**：按住任务左侧的六个点拖拽，调整执行顺序
- **展开设置**：点击任务右侧的 **>**，可以对该任务做更详细的设置
