# 功能介绍

MAK 目前提供以下功能，每项功能都可以独立使用。

::: tabs#ui-type

@tab MXU 版
你可以在 MAK 主界面点击「**添加任务**」，然后点击你需要的功能。MAK 会按照任务列表从上到下的顺序依次执行。

@tab MFAA 版
你可以在 MAK 主界面点击左侧任务列表中的「**添加任务**」，然后选择你需要的功能。MAK 会按照任务列表从上到下的顺序依次执行。
:::

## 如何使用功能？

每个功能称为一个「任务」。具体操作方式会随界面不同而略有区别，你可以：

::: tabs#ui-type

@tab MXU 版

- **勾选/取消勾选**：决定要不要执行这个任务
- **拖拽排序**：按住任务左侧的六个点拖拽，调整执行顺序
- **展开设置**：点击任务右侧的 **>**，可以对该任务做更详细的设置

@tab MFAA 版

- **勾选/取消勾选**：决定要不要执行这个任务
- **调整顺序**：直接在「**任务列表**」拖拽任务，即可调整执行顺序
- **展开设置**：点击选中任务，就可以在右侧的任务设置中对该任务做更详细的设置
:::
