# 领取奖励

让 MAK 帮你自动领取游戏中的各种奖励。

---

## 这个功能会做什么？

当你使用「领取奖励」功能时，MAK 会自动帮你领取以下奖励：

### 1. 任务奖励

- **每日任务奖励**：每天完成日常任务后可以领取的奖励
- **每周任务奖励**：每周完成周常任务后可以领取的奖励
- **军旅成就奖励**：达成成就后可以领取的奖励

### 2. 资源派遣

- **领取派遣奖励**：自动收取已经完成的资源派遣
- **再次派遣**：领取后自动开始新一轮派遣，不用每次都手动操作

---

## 如何使用？

如果你只想领取部分奖励，可以展开「领取奖励」的设置项：

1. 在任务列表中，点击「领取奖励」右边的 **>** 展开详细设置
2. 你可以分别控制以下选项：

   | 设置项 | 说明 |
   |-------|------|
   | **领取任务奖励** | 总开关，关了就不领任何任务奖励 |
   | 　↳ 每日任务 | 单独控制每日任务奖励 |
   | 　↳ 每周任务 | 单独控制每周任务奖励 |
   | 　↳ 军旅成就 | 单独控制成就奖励 |
   | **资源派遣** | 是否领取派遣奖励 |
   | 　↳ 再次派遣 | 领取后是否自动再次派遣 |

3. 根据你的需要，打开或关闭对应的开关
4. 点击「**开始**」

---

## 注意事项

- 使用前请确保游戏已经**登录**并进入了主界面
- 建议配合「[启动游戏](./start-game.md)」功能一起使用，MAK 会先启动游戏再领奖励
- 运行过程中**不要操作模拟器**，让 MAK 自己完成
- 如果有弹窗需要确认，MAK 会自动处理，你不需要手动点击
