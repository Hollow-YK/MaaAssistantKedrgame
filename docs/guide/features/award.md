# 领取奖励

让 MAK 帮你自动领取游戏中的各种奖励。

---

## 这个功能会做什么？

当你使用「领取奖励」功能时，MAK 会自动帮你领取以下奖励：

### 1. 任务奖励

- **每日任务奖励**：每天完成日常任务后可以领取的奖励
- **每周任务奖励**：每周完成周常任务后可以领取的奖励
- **军旅成就奖励**：达成成就后可以领取的奖励

### 2. 资源派遣

- **领取派遣奖励**：自动收取已经完成的资源派遣
- **再次派遣**：领取后自动开始新一轮派遣，不用每次都手动操作

---

## 如何配置？

如果你只想领取部分奖励，可以展开「领取奖励」的设置项：

| 设置项 | 说明 |
|-------|------|
| **领取任务奖励** | 总开关，关了就不领任何任务奖励 |
| 　↳ 每日任务 | 单独控制每日任务奖励 |
| 　↳ 每周任务 | 单独控制每周任务奖励 |
| 　↳ 军旅成就 | 单独控制成就奖励 |
| **资源派遣** | 是否领取派遣奖励 |
| 　↳ 再次派遣 | 领取后是否自动再次派遣 |

---

## 注意事项

- 使用前请确保游戏已经登录游戏
- 运行过程中**不要操作模拟器**
