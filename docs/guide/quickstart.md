这篇指南会手把手教你如何安装和使用 MAK（雪松小助手）。**即使你完全不懂电脑，只要跟着步骤一步一步来，也一定能成功！**

---

## 第一步：检查你的电脑

在开始之前，请确认你的电脑满足以下条件：

1. 你的电脑系统是 **Windows 10 或 Windows 11**（点我查看：[如何查看 Windows 版本](https://support.microsoft.com/zh-cn/windows/查看你的-windows-版本-628bec99-476a-2c13-5296-9dd081cdd808)）
2. 你的电脑上安装了**模拟器**（用来在电脑上玩手机游戏的软件）

::: info 什么是模拟器？
模拟器就是一个软件，让你能在电脑上运行手机游戏。推荐使用以下模拟器：

- **MuMu 模拟器**
- **雷电模拟器**

另外，理论上也支持任何提供了 ADB 连接的模拟器，甚至是真机。

如果你还没装模拟器，先在搜索引擎搜索相应模拟器，下载安装后再继续。
:::

::: note
如果你想要通过真机使用 MAK ，可以参考[ADB设备](./advanced/adb-device.md)
:::

---

## 第二步：下载 MAK

1. 打开浏览器（就是你现在看这个网页用的软件）
2. 在地址栏输入以下网址，然后按回车键：

   ``` text
   https://github.com/Hollow-YK/MaaAssistantKedrgame/releases
   ```

3. 你会看到一个下载页面，找到最新版本（排在最上面的那个），你会看到两种下载方式：

   | 文件类型 | 文件名示例 | 推荐程度 |
   |---------|-----------|---------|
   | **安装包版** | `MAK-win-x64-xxx-Setup.exe` | ⭐ 推荐 |
   | **压缩包版** | `MAK-win-x86_64-xxx.zip` | 适合不想安装的用户 |

---

## 第三步：安装 MAK

::: tabs#install-type

@tab 安装包版

推荐新手使用该方式。

1. 点击下载 `MAK-win-x64-xxx-Setup.exe` 文件
2. 下载完成后，**双击**这个文件（用鼠标左键快速按两下）
3. 如果电脑弹出蓝色的警告框，点击「**仍要运行**」或「**更多信息**」→「**仍要运行**」
4. 按照安装向导的提示，一路点击「**下一步**」，最后点击「**安装**」
5. 等待安装完成

    ::: tip 提示
    部分杀毒软件可能会拦截安装，这是误报，请放心。如果被拦截了，在杀毒软件中选择「**允许**」或「**信任**」即可。
    :::
@tab 压缩包版

1. 点击下载 `MAK-win-x86_64-xxx.zip` 文件
2. 下载完成后，解压至一个单独的文件夹。

    ::: warning 注意
    请使用**纯英文路径**，除非你有能力解决不使用纯英文路径所带来的问题。

    比如解压到 `D:\MAK` 是可以的，但解压到 `D:\我的软件\MAK` 就不行。

    如果你使用的是不推荐的目录，MAK 及开发者有权拒绝为你遇到的问题提供帮助。
    :::

3. 解压完成后，打开解压出来的文件夹，找到 `MAK.exe`，**双击**运行

:::

## 第四步：设置模拟器与游戏

在使用 MAK 之前，请确保你的模拟器和游戏设置正确。

### 模拟器分辨率设置

MAK 通过识别屏幕上的图像来操作游戏，因此模拟器的分辨率非常重要：

1. 打开你的模拟器
2. 进入模拟器的「**设置**」→「**显示**」或「**分辨率**」
3. 将分辨率设置为 **16:9** 的比例（**目前仅支持 16:9，其他比例无法使用**）
4. 支持以下分辨率：

   | 分辨率 | 说明 |
   |-------|------|
   | `1280×720`（720P） | ⭐ 推荐 |
   | `1920×1080`（1080P） | ⭐ 推荐 |
   | 其它 16:9 的分辨率 | 不推荐，可能出现问题 |

::: caution 非常重要
分辨率必须在模拟器的**系统设置**中修改，而不是游戏内的设置。如果模拟器分辨率不是 16:9，MAK 将无法正常工作。
:::

---

## 第五步：初次使用 MAK

1. **启动 MAK**：
   ::: tabs#install-type

   @tab 安装包版
   在桌面找到 `MAK` 图标，双击打开

   @tab 压缩包版
   打开解压的文件夹，双击 `MAK.exe`
   :::

2. **添加任务**：打开 MAK 后，点击「**添加任务**」按钮，选择你想要 MAK 帮你完成的事情

3. **连接模拟器**：根据 MAK 的提示，选择你的模拟器。MAK 会自动尝试连接

4. **开始运行**：点击「**开始**」按钮，MAK 就会开始工作了！
