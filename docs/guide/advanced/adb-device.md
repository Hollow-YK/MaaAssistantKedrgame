title: ADB 设备

ADB（Android Debug Bridge）是 Android 调试桥的缩写，MAK 通过 ADB 与模拟器或真机进行通信。

## 连接真机

如果你希望直接在手机上使用 MAK，需要先通过 ADB 连接你的设备。

::: caution 危险功能
通过真机使用 MAK 属于极度危险的行为，你应确保知道自己在干什么！

:::

### 前置条件

1. **开启开发者选项**：在手机「设置」→「关于手机」→ 连续点击「版本号」7 次
2. **开启 USB 调试**：「设置」→「开发者选项」→ 打开「USB 调试」
3. **使用数据线**将手机连接到电脑

### 连接步骤

1. 打开 MAK
2. 在设备选择界面，选择 **ADB 设备**
3. 如果列表中未出现你的设备，可以尝试手动输入 ADB 地址（默认 `127.0.0.1:5555`）

### 常见问题

- **设备未显示**：检查 USB 调试是否已开启，或尝试重新插拔数据线
- **连接失败**：部分手机需要安装手机厂商的 USB 驱动程序
- **无线连接**：手机和电脑处于同一网络时，可使用 `adb connect <IP地址>` 进行无线调试
