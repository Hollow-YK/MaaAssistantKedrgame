# 问题排查手册

使用过程中遇到问题？先按下面的步骤排查一下，说不定就能解决了！

> 💡 也可以前往 [常见问题（FAQ）](./faq.md) 查看概念性解答。

---

## 运行问题

### 无法启动

::: details 弹窗提示 `Could not find the WebView2 Runtime.`

缺少 WebView2。

前往 [Microsoft Edge WebView2](https://developer.microsoft.com/zh-cn/microsoft-edge/webview2) 下载并安装 `Microsoft Edge WebView2`。

:::

::: details 启动 MAK 时弹出"用户账户控制"窗口

这是 Windows 的安全机制。MAK 需要管理员权限才能操控模拟器窗口。

**解决方法**：直接点击「**是**」即可。每次启动 MAK 都会弹出这个提示，这是正常现象，请每次都点「是」。

> 如果你不希望每次都弹出，可以搜索"Windows 关闭 UAC"了解如何关闭，但**不推荐这样做**，会降低电脑安全性。
> 
> 这相当于你对你家本来有人要来就向你汇报的看门大爷说以后不用汇报直接允许所有人进来一样。

:::

::: details 下载的安装包打不开，提示"Windows 已保护你的电脑"

这是 Windows 的安全提示，因为 MAK 没有签名证书。**MAK 是安全的开源软件**，请放心使用。

**解决方法**：点击弹窗中的「**更多信息**」，然后点击「**仍要运行**」就可以了。

:::

::: details 杀毒软件把 MAK 删掉了

部分杀毒软件会把 MAK 误判为病毒（误报）。

**解决方法**：

1. 打开你的杀毒软件
2. 找到「隔离区」或「恢复区」
3. 找到被删掉的 MAK 文件，选择「**恢复**」并「**添加信任**」
4. 如果不会操作，可以搜索"XX杀毒软件 如何添加信任"

> 如果你不放心，MAK 的全部源代码都在 [GitHub](https://github.com/Hollow-YK/MaaAssistantKedrgame) 上公开，任何人都可以检查。

:::

::: details 解压时说路径错误

压缩包版必须解压到**纯英文**的路径下。

- ✅ 正确：`C:\MAK`、`D:\Games\MAK`
- ❌ 错误：`C:\我的软件\MAK`、`C:\游戏\雪松小助手`

:::

### 无法开始任务

::: details 点击"开始"后什么都没发生

请检查以下几点：

1. 模拟器是否已经启动？（模拟器必须已经启动）
2. 游戏是否已安装在模拟器中？
3. 是否在 MAK 中添加了至少一个任务？
4. 启动 MAK 时是否点了「是」（管理员权限）？如果没有，关闭 MAK 重新以管理员身份运行

:::

::: details 不支持当前控制器

在 MAK 右上角的「连接设置」中选择合适的控制器。

:::

::: details 无法连接设备

- 确保你打开了模拟器或连接了设备
- 确保你是**以管理员身份运行**的 MAK

:::

::: details 资源加载失败

尝试按你的安装方式重新获取文件：

    ::: tabs#install-type

    @tab 安装包版

    在「设置」→「应用」→「安装的应用」中找到 MAK，选择「卸载」，然后重新安装。

    @tab 压缩包版

    删除 MAK 所在目录下全部内容（可以保留 `config` 文件夹），然后重新解压。

    :::

:::

::: details 出现 `HD_python.exe` 窗口

你的电脑可能存在**蠕虫病毒**，建议尽快杀毒并重装系统。我们不受理该类问题。

:::

### 任务运行异常

::: details 为什么 MAK 总是识别失败？

可能的原因：

1. **模拟器分辨率不对**：模拟器的分辨率必须设置为 16:9，推荐 1280×720 或 1920×1080。请在**模拟器的系统设置**中修改（不是游戏内设置）。其他比例**目前不支持**。
2. **没有以管理员身份运行**：MAK 需要管理员权限，启动时看到"用户账户控制"弹窗请点「是」
3. **使用的是老版本**：MAK 仅会为最新版提供支持，一般不会考虑老版本。
4. **游戏进行了更改**：如果游戏版本更新改变了界面，MAK 可能需要一段时间才能适配。

:::

::: details 无法正常进行点击/操作

- 确保你的 MAK 处于**纯英文**路径下，并且没有全角符号
- 确保你是**以管理员身份运行**的 MAK
- 确保 MAK 连接了正确的设备
- 尝试重新安装

:::

::: details 任务频繁失败

可能是我们的 Bug 导致的，也可能是《雪松》的

:::

## 各任务相关问题

### 启动游戏

::: details 无法启动游戏

请检查：

1. 模拟器分辨率是否为 16:9（推荐 1280×720 或 1920×1080）
2. MAK 是否以管理员身份运行
3. 游戏是否已正确安装在模拟器中
4. 是否选择了正确的资源文件，并确保是我们支持的渠道服之一

:::

### 领取奖励

::: details 无法领取奖励

确保你已登录游戏，且奖励可以正常手动领取。如果游戏界面有更新，MAK 可能需要一段时间来适配。

:::
